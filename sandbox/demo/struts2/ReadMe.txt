struts2演示

1./helloworld - helloworld
2./spring     - 与spring整合
3./coc        - 惯例优先配置，零配置文件                   - codebehind不支持redirect，chain等操作，需要自己写jsp跳转
4./crud       - CRUD，Create Read Update Delete原子操作
5./crud2      - 整合spring, hibernate实现的CRUD操作
6./wildcard   - 使用通配符，替代零配置                     - 通配符似乎不支持actionName!methodName类型的动态方法调用
7./fileupload - 文件上传
